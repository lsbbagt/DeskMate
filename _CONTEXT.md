## 🔑 关键决策记录

### 2026-03-29
1. **项目重构决策：** 从精简版本开始
   - 只保留核心功能
   - 确保稳定后再扩展
   - 避免之前bug多的问题

2. **收藏夹MVP范围：** 单标签页，基础CRUD
   - 暂不实现多标签、多层文件夹
   - 专注核心流程稳定

3. **技术选型：** 保持与v1一致
   - Wails + Vue3 + Vuetify3
   - 浅蓝色主题（#5B9BD5）

4. **标题栏实现演进：**
   - v1: Vuetify v-app-bar + CSS app-region（失败）
   - v2: 原生HTML + Wails标准拖动（`--wails-draggable:drag`）
   - 原因：Vuetify组件与Wails拖动机制有冲突

5. **打开方式设计：**
   - 添加收藏时可选择内部/外部打开
   - 列表中显示当前打开方式
   - 可快速切换打开方式
   - 部分网站（GitHub等）建议使用外部打开

6. **数据持久化方案：**
   - 存储位置：`~/.statbox-v2/`
   - 文件格式：JSON
   - 后端API：`SaveBookmarks()` / `LoadBookmarks()`

## 🐛 已解决的问题

1. ✅ **包名冲突** - app.go 和 main.go 包名不一致导致编译失败
2. ✅ **窗口控制按钮无效** - 使用错误的 API（WindowClose），应使用 Quit
3. ✅ **模板文件编译错误** - HelloWorld.vue 引用不存在的 Greet 方法
4. ✅ **GitHub等网站打不开** - iframe限制，改为用户选择内部/外部打开
5. ✅ **标题栏显示问题** - 移除density="compact"，强制设置高度
